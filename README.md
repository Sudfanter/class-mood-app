# Mood Tracker iOS App

This is a simple Mood Tracker app built using SwiftUI.

## Features
- Emoji-based mood logging
- Time-stamped mood entries
- Clean, beginner-friendly UI

## How to Run
1. Clone the repository
2. Open `lucasclassapp` in Xcode
3. Select an iOS Simulator
4. Click Run
